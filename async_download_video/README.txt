----------------------------------------Краткое описание настройки и использования модуля загрузки видео---------------------------------------------------------------

Для использования данного модуля загрузки видео необходим настроенный стек LAMP, библиотека jquery не ниже 1.7, подключенный модуль Mysqli для Mysql;

------------Структура------------------
1. video.sql - стурктура базы данных для добавления информации о загружаемом видео;
2. jquery.downloadVideo.js - модуль на jquery для представления формы с кнопкой для выбора видео и отправкой видео на сервес в составе других полей формы;
3. class_download_video.php - класс на php для осуществляния взаимодействия с базой данных. Передача данных из формы (jquery.downloadVideo.js) на сервер, запись в папку/путь в БД;
4. upload_video.php - промежуточный файл, осуществляющий взаимодействие между jquery.downloadVideo.js и class_download_video.php по средствам передачи данных (POST) от фунцкций jquery.downloadVideo.js в функции class_download_video.php. Максимальное количество видео, которое может записать один пользователь 4, соотвественно перед записью осуществляется проверка на количество записей в 

------------Использование------------
На странице, в блоке кода jquery $(document).ready (function() { ...............}), привяжите фунции downloadVideo('addFormVideo'), downloadVideo('addInputVideo',$("form" модуля jquery.downloadVideo.js к желаемому событию, например щелчок по ссылке:
divLeft.on('click', '#addVideo', function() {
  	                                             divMiddle.empty();
  	                                             divMiddle.downloadVideo('addFormVideo'); //функция добавления формы в див
  	                                             divMiddle.downloadVideo('getAllRezident',$("form")); //функция добавления селекта, для выбора id резидента
  	                                             $("form").downloadVideo('getCategoryVideo'); //функция вывода категорий видео
  	                                             divMiddle.downloadVideo('addInputVideo',$("form")); //функция добавления инпута для выбора видео и кнопки для отправки данных формы
                                               });
